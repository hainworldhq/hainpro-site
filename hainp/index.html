HAIN Protocol (HAINP)

HAINP is the policy-aware execution and interoperability layer for hybrid AI-native systems across local, edge, private, and cloud environments.

What HAINP is

HAINP is designed to define shared semantics for execution domains, locality constraints, routing intent, governed fallback, context portability, and trace continuity.

What HAINP is not

HAINP is not a transport protocol, not a replacement for MCP, and not a replacement for A2A. It is intended to complement the existing protocol stack by defining the missing execution and governance layer.
